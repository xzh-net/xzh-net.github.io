# Sacal 2.12.16

## 1. Windows

### 1.1 安装SDK

下载地址：https://www.scala-lang.org/download/2.12.16.html

设置环境变量

```lua
SCALA_HOME
D:\scala\scala-2.12.16

PATH
%SCALA_HOME%\bin
```

> cmd窗口执行 `scala`
