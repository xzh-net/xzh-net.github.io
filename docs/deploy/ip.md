
# IP子网划分



| 类别    | 首字节范围 | 二进制前缀 | 网络位长度 | 默认掩码            |
| :------ | :--------- | :--------- | :--------- | :------------------ |
| **A类** | 1-126      | `0xxxxxxx` | 8位        | 255.0.0.0 (/8)      |
| **B类** | 128-191    | `10xxxxxx` | 16位       | 255.255.0.0 (/16)   |
| **C类** | 192-223    | `110xxxxx` | 24位       | 255.255.255.0 (/24) |
| **D类** | 224-239    | `1110xxxx` | -          | 组播地址            |
| **E类** | 240-255    | `1111xxxx` | -          | 保留地址            |
